Find where your sources
disagree.

Standard RAG retrieves from a single document and answers confidently — even when two of your policies silently contradict each other. Crosscheck inverts this: it audits your entire knowledge base, surfaces every conflict, cites both sides, classifies the clash type, and ranks by severity. Grounded via Microsoft Foundry IQ.

Step 01
Retrieve across sources

Foundry IQ pulls relevant passages from many documents simultaneously, each with a grounded citation.

Step 02
Extract & compare claims

Each passage is decomposed into atomic claims; the agent compares claims that address the same subject.

Step 03
Report cited conflicts

A ranked conflict report — every contradiction requires two independent citations from two different sources.

    Setup & access

    Go live on Foundry IQ — 3 commands
    1. Check access (read-only): ./infra/check_access.ps1
    2. Provision + upload corpus: ./infra/setup_foundry_iq.ps1
    3. Create knowledge base in ai.azure.com → Build → Knowledge, then set PROVIDER=foundry in .env and restart.

    An empty Azure subscription is expected — the setup script uploads corpus/ for you. Full guide: infra/README.md.
